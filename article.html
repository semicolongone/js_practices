Learning HTML

HTML stands for HyperText Markup Language. It is the basic building block used to create webpages. Every website on the internet uses HTML to structure its content.

Why Learn HTML?

  • It is the foundation of web development
  • It is easy to learn
  • It works with CSS and JavaScript

      • Once you understand HTML, you can start building your own websites and web applications.
